# Valentine Yes/No Surprise

A playful, romantic single-page React site with a 3D-styled UI, animated glow background, and the classic "Will you be my Valentine?" moment. The "No" button grows the "Yes" button for four tries, then becomes evasive on the fifth attempt so the only winning move is "Yes".

## Run locally

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Add your GIFs

Place three GIF files in `public/gifs/` named `love-1.gif`, `love-2.gif`, and `love-3.gif`. You can replace those names in the markup inside the app if you want different filenames.
